# Структура папки lib/

```
lib/
├── core/           # Цвета, текстовые стили, константы, утилиты
├── data/           # Firebase, модели, репозитории
├── logic/          # BloC, Cubit и другая бизнес-логика
├── screens/        # UI-экраны (авторизация, меню, ветеринары, новости и т.д.)
├── widgets/        # Переиспользуемые компоненты
├── routes/         # AppRouter (настройки навигации: Navigator или GoRouter)
└── main.dart       # Главный файл приложения
```
